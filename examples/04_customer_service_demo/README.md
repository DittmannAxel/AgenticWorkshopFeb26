# Example 04: Full Customer Service Demo

Interactive customer service voice agent demo with text console.

## What it does

This is the most complete example — an interactive session where you type customer messages (simulating speech) and the agent responds using all available tools.

## Run

```bash
python examples/04_customer_service_demo/main.py
```

## Sample Session

```
Kunde: Guten Tag, ich habe eine Frage zu meiner Bestellung.
Agent: Guten Tag! Ich helfe Ihnen gerne. Können Sie mir Ihre Kundennummer nennen?

Kunde: Meine Kundennummer ist C-1001.
Agent: Vielen Dank, Frau Schmidt. Ich sehe zwei Bestellungen...

Kunde: Was ist mit ORD-5001?
Agent: Ihre Bestellung ORD-5001 befindet sich im Versand und wird morgen zwischen 10-14 Uhr geliefert.

Kunde: Ende
Agent: Vielen Dank für Ihren Anruf. Auf Wiederhören!
```
